# Kismat Jewellery

A modern, elegant silver jewelry e-commerce frontend built with Next.js 14+, TypeScript, and Tailwind CSS.

## Features

- Modern, responsive design
- AI-powered Soul Stone Matcher (Gemini API)
- AI Concierge Chat (Gemini API)
- Hero carousel with auto-rotation
- Mobile-responsive navigation
- Product showcase

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file and add your Gemini API key:
```
NEXT_PUBLIC_GEMINI_API_KEY=your_api_key_here
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript (Strict mode)
- Tailwind CSS
- Lucide React (Icons)
- Google Gemini API
